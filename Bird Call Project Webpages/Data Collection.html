Data Collection

Pulling the Data:

  • Found a convenient API through Xeno-Canto - an online repository dedicated to sharing bird sounds from around the world.

  • Query parameters limited to the United States for Country and Oregon for State for the scope of this project. (Figure 1)

  • We were able to successfully return a total of 763 recordings from 160 unique bird species. Each object in the recordings array were returned in the following format: (Figure 2)

  • A For loop was created to pull the following information for each recording and append it to lists that will be used to store data into a dataframe (Figure 3). An additional Column “Recording Name” was created in the dataframe which took the id of the recording and appended “.mp3” at the end. (Figure 4). This dataframe was exported as a CSV.

Cleaning the Data:

  • A group by and count function was used to identify bird species that only had 1 recording and dropped these species from the dataframe (Figure 5). This served as our training set for the machine learning model. (78 unique species and 300+ recordings).

  • A For loop was created to download the actual recordings that will be fed into the machine learning mode to train and evaluate accuracy. (Figure 6)

Challenges:

  • All bird species did not have multiple recordings- identified and removed.

  • Linking metadata to recordings- included Recording Name in CSV

  • Eventually found that .mp3 files were a bit more difficult to work with in the librosa library. Conversion of the .mp3s to .wav files needed, connected with Iris to complete.
