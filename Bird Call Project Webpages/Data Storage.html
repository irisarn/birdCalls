Data Storage

Using Pandas and SQLite

  • Using pandas, the data that Jasmin had collected via the API was transferred into an SQLite database for ease of use on the project webpage.

    • The database has three tables:

    • Metadata: Metadata information concerning the collected recordings

    • Recordings: Information pertinent to each specific recording

    • Unique Names: List of pictures of the birds spotted in Oregon

  • The database was created using a jupyter notebook.

    • The commands are recorded in: “Create_birdDB.ipynb”.

  • I scraped the pictures included in the database using code adapted from the article "Image Scraping with Python" published on https://towardsdatascience.com/ by Fabian Bosler.

    • The code is included in the jupyter file “search&download bird pictures.ipynb”.

    It consists of three functions:

    • the first one scrapes the data using a google search

    • the second one downloads the pictures

    • the third one drives the two previous functions providing the information that needs to be searched.

  • I also created a google search for other bird specific sites to augment the information provided by our webpage.

    • To do so I used the information included in https://developers.google.com/custom-search/v1/overview

    • crated an API key and used the control panel to specify our custom search.

    • I then played around with the Look and Feel options to design the search that best would adapt to our webpage design.
